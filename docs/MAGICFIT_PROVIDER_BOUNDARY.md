# MagicFit Provider Boundary

MagicFit is a candidate render provider only.

Allowed:
- Black Ledger Newsroom B-roll
- Faction promo scenes
- Short photoreal anchor tests
- Social video derivatives
- Text-to-video/image-to-video provider bake-off

Forbidden:
- direct publish
- editorial truth
- product behavior proof
- private campaign data
- sourcebook text
- unreviewed public videos

MagicFit may create candidate assets only after provider verification and QA receipts exist.
It may not publish to `chummer.run`, set editorial truth, or become product proof.
